ArrayList<>集合常用方法

Java中的集合可以存储大部分数据类型，但是以下几种数据类型是不能直接存储在集合中的

  1. 基本数据类型：例如int、double、char等。因为集合只能存储对象类型，而基本数据类型不是对象类型，需要通过装箱操作将其转换为对应的包装类，例如Integer、Double、Character等。

  2. void类型：void表示没有返回值的方法，不属于任何数据类型。

  3. 数组类型：数组是一种复合类型，不属于基本数据类型或对象类型，因此不能直接存储在集合中。

  4. 泛型类型中的基本数据类型：例如ArrayList，虽然Integer是一个对象类型，但在泛型类型中仍然不能存储基本数据类型，需要使用对应的包装类。

需要注意的是，虽然Java中的集合可以存储大部分数据类型，但在实际使用时还需要考虑到数据类型的特点和需求，选择合适的集合类型和存储方式。

面向对象…

类就是描述对象的组成的，类是由属性和方法组成的。类可以说是设计一个对象的构成代码；对象就是类的实例。

1. 类的基础
  • 创建类
  • 创建对象
  • 访问对象的属性
  • 访问对象的方法
  • 创建构造方法(构造方法可以理解为实例化对象是最开始执行的方法)
  • static修饰符

被static修饰的成员变量被称为静态变量

被static修饰的成员方法被称为静态方法

静态方法不能访问非静态变量的非静态方法

非静态方法可以访问所有

在类中使用了static修饰符，那么这个方法或者属性就是每个对象共享的。比如说在一个班上有一个学生类，其中类中的属性有一个是’’老师’’，那么这个属型存在一个特性，就是每个学生对象里面的’老师’属性都是一样的，在每次创建一个学生对象是都需要对‘老师’属性进行重复的设置，如果将‘老师’这个属性的修饰符这是成static静态，那么我们在创建第一个学生对象时，将‘老师’属性设置值后，那么在创建其他学生对象就不需要在设置了，因为‘老师’属性设置了static修饰符，它是在每个创建的对象里面共享的。方法也如此。

2. Java的几种类
  • 测试类

测试类就是有main方法存在，测试类一般用来调用其他的类用来测试，示例

  • JavaBeen类

JavaBeen类主要是用来描述一种对象所创建的类，比如学生对象我们可以写成一个类，属性有名字和年龄，方法有学习。JavaBeen类还需要提供空参构造方法和实参构造方法，对每一个属性需要设置成私有属性，不能让外部直接访问到属性，要通过提供对应的get和set方法访问属性。如下模板

  • 抽象类(工具类)

抽象类就是不能用来描述一种对象的类，比如说一种求平均数的类，那不能说是对象，只能创建一种抽象类。抽象类特点是不能创建对象，所有的的属性和方法都是需要添加静态修饰符。使用抽象类的方法时直接类名加方法。
